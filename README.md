## 📌 About
An end-to-end AI-powered career intelligence system that analyzes resumes and matches them with 10,000+ LinkedIn job listings using NLP and Machine Learning.

## ✨ Features
- 📄 **Resume Parser** — Extracts skills, email, phone from PDF automatically
- 🎯 **AI Job Matching** — Matches resume with 7,000+ jobs using TF-IDF & Cosine Similarity
- 📊 **Skill Gap Analysis** — Identifies missing skills based on market demand
- 📚 **Learning Roadmap** — Suggests free resources for missing skills
- 🌐 **Interactive Dashboard** — Built with Streamlit
